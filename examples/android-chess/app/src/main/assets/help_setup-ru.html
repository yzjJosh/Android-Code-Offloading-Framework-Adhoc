Расстановка позиций на доске

Коснитесь одной из фигур внизу доски для того, чтобы выбрать её, и затем коснитесь квадрата, в котором вы хотите её поставить.
Чтобы удалить фигуру, выберите её и нажмите на кнопку удаления.
Чтобы переместить фигуру, выберите её и коснитесь квадрата, в который вы хотите её переместить.
Для изменения цвета новой фигуры выберите квадрат рядом с X.
Когда вы закончите, установите флажок.
