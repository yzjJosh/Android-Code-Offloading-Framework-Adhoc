Помощь в тренировке

Найдите матовую комбинацию (может быть более одного хода).
Ваше время и среднее время для каждой позиции указано рядом с доской.
Нажмите кнопку 'Показать' для решения.
Нажмите кнопку 'Далее', чтобы перейти к следующей позиции.
Иногда может быть найдено несколько решений, но приложение принимает только одно их них.

Это приложение содержит 2000 последовательностей вынужденного мата, взятых из игр гроссмейстеров.
