Esporta il database di gioco nella scheda SD

Tutte le partite nel database possono essere esportate nella scheda SD in un singolo file PGN. Questo file verrà denominato chess.pgn e un eventuale file gi√† esistente verrà sovrascritto.

Importa le partite nel database di gioco

Usa questa opzione per aggiungere partite al database di gioco. L'opzione permette di navigare a un file PGN. Selezionalo per importarlo.

Cancella tutte le partite dal database di gioco

Usa questa opzione se vuoi ripulire il database di gioco. Assicurati di aver fatto un backup nella scheda SD, poiché l'azione non può essere annullata.

Crea database delle aperture

Naviga fino a un file PGN nella scheda SD e questo verrà convertito in un database delle aperture con lo stesso nome, ma con un nome di file con estensione .bin. Attualmente tutte le partite nel file PGN saranno analizzate fino alla mossa 17.

Imposta database delle aperture

Naviga fino al file .bin appena creato (database delle aperture) per permettere a Scacchi di utilizzarlo.

Installa motore UCI

Seleziona un motore UCI già installato per consentire a Scacchi di utilizzarlo (ad esempio Komodo).

Installa manualmente motore UCI

Naviga fino al file binario del motore UCI per consentire a Scacchi di utilizzarlo.

Disinstalla motore UCI

Elimina il motore UCI installato.

Crea set di pratica

Crea posizioni per la pratica dalle partite terminate con uno scacco matto. Naviga fino al file per elaborarlo. Le partite rifiutate normalmente lo sono perché non finiscono in uno scacco matto o sono duplicati di altre partite
