Помощь при игре онлайн

FICS: Free Internet Chess Server позволяет играть с тысячами шахматных игроков онлайн.
Предупреждение: если ваше устройство переключается с одной точки доступа на другую, соединение может оборваться. Timeseal не используется.

Меню

Вызовы
Кто ищет матч. Нажмите на игрока в списке, чтобы принять вызов.
Искать
Сделайте собственный вызов. Если вы вставите ник знакомого игрока, будет сделан конкретный вызов.
Игры
Смотрите игры, проходящие в онлайн-режиме. Гроссмейстер может быть определён по префиксу GM в имени пользователя.
Игроки
Показывает незанятых игроков с рейтингом.

Кнопки

Две экранные кнопки могут быть использованы для:
- доступа к списку полезных функций
- кнопка для сплетен - позволяет зарегистрированным пользователям общаться с другими наблюдателями игры.
Для просмотра наблюдателей игры напечатайте: allobservers [номер игры]

Исследовать

Пройдитесь по игре с разными вариантами. Используйте кнопку возврата, чтобы вернуться к основному варианту.
Возьмите с собой тренера. Разрешите игроку делать ходы, напечатав: mexamine [игрок]
Используйте телефон с выходом в Интернет для голосового общения (например, Skype)
Когда закончите - напечатайте: unexamine
