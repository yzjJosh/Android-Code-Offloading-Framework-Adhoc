Расширенные

Создавайте или импортируйте собственные материалы.

Экспорт игровой базы данных на SD-карту

Все игры в игровой базе данных могут быть экспортированы в единый PGN-файл на SD-карту. Этот файл будет назван chess.pgn, а уже существующий файл будет перезаписан.

Импорт игр в игровую базу данных

Используйте эту опцию для добавления игр в игровую базу данных. Она позволяет перейти к PGN-файлу. Выберите нужный и выделите его для импорта.

Удалить все игры из игровой базы данных

Используйте эту опцию, если хотите очистить игровую базу данных. Убедитесь, что вы сделали резервную копию на SD-карте, это действие не может быть отменено.

Создать базу данных дебютов

Перейдите к PGN-файлу на SD-карте, и он будет сконвертирован в базу данных дебютов с тем же именем, но с расширением .bin. В настоящее время все игры в PGN-файле обрабатываются до 17-го полухода.

Установить базу данных дебютов

Перейдите к только что созданному .bin-файлу базы данных дебютов, чтобы разрешить Шахматам использовать его.

Установить движок UCI

Выберите установленный движок UCI, чтобы разрешить Шахматам использовать его (например, Komodo).

Установить движок UCI вручную

Перейдите к двоичному файлу движка UCI, чтобы разрешить Шахматам использовать его.

Удалить движок UCI

Удалить установленный движок UCI.

Создать тренировочный набор

Создаются тренировочные позиции из игр, заканчивающихся матом. Перейдите к файлу для обработки. Неудачные записи обычно происходят для игр, которые не заканчиваются матом или которые дублируют уже имеющиеся игры.

Сбросить тренировочный набор

Хотите начать тренировку сначала? Это также сбросит счётчик вашего среднего времени.
