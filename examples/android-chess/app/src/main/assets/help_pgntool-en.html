Advanced

Create or import your own stuff here.

Export game database to sd-card

All games in the game database can be exported to a single PGN file on sd-card. This file will be named chess.pgn and an existing file will be overwritten.

Import games into the game database

Use this option to add games to the game database. It lets you browse to a PGN file. Select and choose to import.

Delete all games in the game database

Use this option if you want to clean up the game database. Make sure you first make a backup to sd-card, this action can not be undone.

Create openingdatabase

Browse to a PGN file on sd-card and it will be converted into an openingdatabase with the same name but with file name extension .bin. Currently all games in the PGN file will be parsed until 17 ply.

Set openingdatabase

Browse to the newly created .bin openingdatabase file to let Chess use it.

Install UCI engine

Select an installed UCI engine to let Chess use it (e.g. Komodo).

Manually install UCI engine

Browse to a UCI engine binary file to let Chess use it.

Uninstall UCI engine

Remove the installed UCI engine.

Create practice set

Creates practice positions from games that end in a check-mate. Browse to the file to process. Failed entries are usually games that did not end in a check-mate or are a duplicate

Reset practice set

Want to start over practicing? This will also reset your average time counter.
